# 🚀 Task Management System API

Hệ thống API Quản lý User, Project và Task. Ứng dụng phân quyền chặt chẽ bằng Spring Security & JWT, hỗ trợ chuẩn hóa cấu trúc dữ liệu và xử lý ngoại lệ tập trung.

## 🛠 Công nghệ sử dụng
- **Java 17** / **Spring Boot 3.x**
- **Spring Data JPA** / **Hibernate**
- **Spring Security & JWT** (Authentication & Authorization)
- **Database:** SQL Server
- **Documentation:** Swagger (OpenAPI 3)
